About EcoTracker

EcoTracker represents the culmination of modern web development principles applied to solve real-world environmental challenges. This college project demonstrates proficiency in HTML5, CSS3, and responsive design while addressing the critical need for sustainability awareness in educational institutions.

Built entirely with semantic HTML and advanced CSS techniques, the project showcases clean code architecture, accessibility best practices, and mobile-first design principles. The platform serves as both a functional tool and a demonstration of technical expertise.

Technologies Used
